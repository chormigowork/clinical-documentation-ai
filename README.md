# Clinical Documentation AI

**End-to-End AI Pipeline for Automated Clinical Documentation**

Clinical Documentation AI is a portfolio project that explores how Generative AI can assist healthcare professionals by transforming structured clinical information into draft clinical documentation.

This project is inspired by my Final Degree Project in Applied Data Science, which received a final grade of **9.1/10**.

## Project Status

🚧 Work in progress — repository currently being prepared for publication.

## Objectives

- Process structured clinical data
- Build optimized prompts for clinical text generation
- Generate draft discharge summaries using Generative AI
- Apply post-processing to improve consistency
- Document the complete workflow as a reproducible AI pipeline
